# 中国房地产趋势预测 Skill / China Real Estate Trend Forecast Skill — 使用说明

这是一个轻量级中国楼市预测 Skill，用于辅助判断新房、二手房、核心区、远郊、新区和卫星城的价格趋势、风险和概率。它保留多 Agent、评分模型、概率预测、Bear Case Review、自住/投资分离等能力，并新增“最新市场数据检索层”。

它不是爬虫系统、宏观经济论文系统或机器学习系统。它只使用公开可访问数据；不能绕过付费墙、验证码、登录限制或反爬机制。

免责声明：仅供交流学习娱乐，不提供投资建议，所有解释权归作者所有。

## Mode 1: User-Provided Data Mode

用户提供数据，Skill 负责评分、交叉验证和预测。

适合：

- 用户已经有数据表
- 用户正在做研究报告
- 用户要检验自己的判断

使用方式：

```text
请使用 china-real-estate-quant-forecast-skill，
基于我提供的数据分析【城市/区域】的【新房/二手房】市场。
预测周期：【3/6/12/24 个月】。
用户目的：【自住/投资/研究/风险检查】。
```

## Mode 2: Fresh Data Retrieval Mode

当用户询问最新走势，且运行环境支持联网搜索时，Skill 主动检索公开最新数据。

适合：

- 用户只提供城市和问题
- 用户问当前走势
- 用户问现在是否适合买房
- 用户问未来几个月走势
- 用户问某城市是否见底

触发表达包括：最新、当前、现在、今年、最近、当下、未来几个月、未来 3 个月、未来 6 个月、未来 12 个月、是否见底、现在是否适合买房、当前是否适合投资。

限制：

- Skill 只能使用公开可访问数据。
- 不绕过付费墙、验证码、登录限制。
- 找不到数据时会标注 Data Gap。
- 自动检索结果仍需做来源评级和新鲜度判断。
- 抓取到的数据不代表绝对准确，必须检查来源、日期、口径和冲突。

## 数据检索注意事项

- 自动检索依赖运行环境是否支持联网。
- 检索结果必须有来源名称和时间。
- 搜索热度、社交媒体和中介反馈只能辅助判断。
- 没有成交量和挂牌量时，短期预测置信度会降低。
- 数据越新不代表越准确，仍需看来源等级和口径。

## 小白板块：第一次怎么用

如果你不熟悉楼市分析，只要告诉 Agent：

1. 城市和区域
2. 新房、二手房，还是两者都看
3. 你是自住、投资、研究，还是做风险检查
4. 你关心 3 个月、6 个月、12 个月还是 24 个月
5. 是否需要最新数据

示例：

```text
现在北京二手房是不是见底了？
请区分自住和投资，输出 3/6/12/24 个月上涨、横盘、下跌概率。
```

小白常见误区：

- 降息不等于房价一定涨。
- 人口流入不等于所有区域都涨。
- 政策放松不等于成交真的恢复。
- 挂牌价不等于成交价。
- 新房备案价不等于真实成交价。
- 自住能买，不代表适合投资。
